| Interface | Description | Mode | VLANs | Native VLAN | Trunk Group | Channel-Group |

| Ethernet1 |  to s2-pe01 - CLIENT use-case - Remote end | trunk | 1811, 2002 | - | - | - |

```eos
!
interface Ethernet1
   description to s2-pe01 - CLIENT use-case - Remote end
```
